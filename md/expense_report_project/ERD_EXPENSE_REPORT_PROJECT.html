Catatan Teknis

⚠ Penting untuk Developer

  • Relasi Polymorphic Manual: Pada tabel detail, field id_detail_kasbon mengacu pada baris (ID) dari tabel detail kasbon, dan direferensikan berdasarkan nilai pada field tipe.
  • Cross-Database Interface: Untuk referensi akun bank, aplikasi terhubung dengan `ms_bank` dari database eksternal. (DB `gl_sendigs` / `sendigs_finance`).
  • File Uploads: Bukti nota disimpan secara riil di folder upload, dan DB hanya mencatat URL / path foldernya saja.
